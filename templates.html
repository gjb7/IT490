Templates

This document describes the templating system used, as well as how to create new new templates.

Creating a Template

Under the hood, Twig is used for our templating engine. All templates are written in Twig's syntax. You should read up on Twig's syntax before continuing with this document.

Templates are usually used in conjunction with a controller. When a method is called on a controller because of a user navigating to a specific URL. The template should have the same name as the method on the controller class, and should be stored in app/views/<controller name>, where the controller name is the controller class, without Controller, and then made lowercase.

When creating a new template, you should extend the _layout.twig template, and override the title and content blocks. For information about those techniques, read the documentation on template inheritance.
